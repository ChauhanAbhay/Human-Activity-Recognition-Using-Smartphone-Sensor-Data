# Human Activity Recognition Using Smartphones

## Overview

This project compares Support Vector Machine (SVM), Random Forest, and Artificial Neural Network (ANN) models for Human Activity Recognition using the UCI HAR Dataset.

The dataset contains smartphone accelerometer and gyroscope measurements collected from 30 volunteers performing six activities:

- Walking
- Walking Upstairs
- Walking Downstairs
- Sitting
- Standing
- Laying

The objective is to classify human activities from sensor data and compare the performance of different machine learning algorithms.
